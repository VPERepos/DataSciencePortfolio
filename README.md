# DataSciencePortfolio
## Motivation
The goal of this collection of projects is to demonstrate the knowledge, experience and ability of the author to solve data science problems in the field of bussiness data analytics and modelling.
## Structure
Each subfolder consists of a corresponding README file, where one can find more details on each project, like problem statement and tools and technologies that have been used. There are also jupyter notebooks representing explorative data analysis and modelling prototypes. Each notebook is filled with comments in order to enable the understanding of conducted steps.
## Transparency and Tooling
The projects in this portfolio were developed with the assistance of AI-based tools (e.g., ChatGPT) for brainstorming, code refinement, and documentation support. All core components, including problem formulation, feature engineering, model selection, evaluation design, and interpretation of results, were independently implemented and critically validated by the author.
