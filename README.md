# DataSciencePortfolio
## Motivation
The goal of this collection of projects is to demonstrate the knowledge, experience and ability of the author to solve the data science problems in the field of the bussiness data analytics and modelling. 
## Transparency and Tooling
This project was developed with the assistance of AI-based tools (e.g., ChatGPT) for brainstorming, code refinement, and documentation support. All core components—including problem formulation, feature engineering, model selection, evaluation design, and interpretation of results—were independently implemented and critically validated by the author.
